# 阿德勒法

想要帮助一个人改正问题,不要告诉他后果怎样,而是为他解决执行中的困难,我们要关注于如何解决问题

健康的自尊是培养出孩子,'我能行'的信念

# 家庭原则草案

  1. 不吃饭不允许吃零食
  2. 自己的东西自己收拾,不收拾就就一周不允许玩
  3. 碰到困难自己解决,大人可以给予解决方案,不允许直接帮助

# 不能做的事情:

  1. 不要给孩子一次性买太多的东西
  2. 不要去包揽孩子遇到的困难,要告诉其解决思路,让他自己去解决困难
  3. 不要替孩子做作业
  4. 不要对孩子发脾气,然后又快速安抚,前后行为不一会让孩子失去安全感

# 需要做的事情

  1. 一周一次的家庭会议
  2. 尊重孩子,父母跟孩子是平等关系,孩子需要话语权
  3. 让孩子跟父母一起共同承担家庭事务,边做边干,边干边学,可以培养下面的感知力跟技能
  4. 遇到问题时缓冲对立面,我们是一个团队\家庭,一同去解决问题达成共识

# 正面教育的拼版

  • 理解孩子错误行为的目的
  • 和善与坚定并行
  • 相互尊重
  • 错误是学习的好机会,惩罚不是目的
  • 社会责任感
  • 家庭会议跟班会
  • 让孩子参与解决问题
  • 给予鼓励

# 当今社会变化以及现状

  • 孩子很少有机会培养责任感跟上进心,他们现在无需像过去那样为了家庭生计付出,相反孩子们被以爱的名字给予的太多而无需付出,使之认为理所当然
  • 父母会包揽孩子遇到的一切困难而剥夺了孩子们自己战胜困难的信念
  • 孩子的动作及话语都是思想的隐射

# 成熟最重要的七项能力

  1. 个人能力的感知 - 我能行
  2. 在团队或者社会关系中的感知 - 社会价值或者团队价值
  3. 自己在生活中的力量或者影响力 - 我能够影响或者改变自身
  4. 自省自律 - 理解个人情绪,并利用这种理解来自我控制
  5. 沟通 共情 合作 分享 倾听
  6. 整体把握力度
  7. 判断能力 - 阅历以及知识面

# 父母的三种类型:

  • 严厉型:这些事你必须准守的规则,违反规则会受到惩罚
  • 骄纵型:没有或者低规则,我们互相爱着对方并且会幸福
  • 正面管教型:一起制定对双方有利的规则,共同决定遇到问题时的解决方案,尊重彼此的决断

# 对立的惩罚会造成五种结果

想想自己上次被羞辱或者惩罚时的感受吧,你有过合作的意向了吗?想要表现的更好了吗?

  1. 愤恨: 这不公平,不能相信大人
  2. 报复: 这回他们赢了,但我会搬回来
  3. 反叛: 我偏要对着干,证明我不是必须按他们要求做
  4. 退缩:
    • 偷偷摸摸: 下次绝对不会被抓到
    • 自卑: 我是个坏孩子,我不行

# 有效管教的四个标准

  1. 是否和善且坚定
  2. 是否有助于孩子感受归属感跟价值感
  3. 是否长期有效
  4. 是否能教给孩子有价值的技能\品格\思想

过度控制 随时捕捉孩子的好坏行为并对之惩罚显然是父母在承担责任,那么,当大人不在场时,孩子显然没法学会对自己行为负责

# 和善与坚定并行

  1. 和善的意义是表达我们对孩子的尊重
  2. 坚定的意义是表达对我们自己及当时环境的尊重
  3. 大人要求孩子控制自己的行为,难道要求自己学会控制自己的行为很过分吗

# 例子

人在生气的时候是很难理想思考的 决定自己怎么做,比让孩子怎么做好好得多

  • 孩子顶嘴:离开战场,等心平气和后再去共同讨论解决方案
  • 碰到问题时,孩子如果习惯了父母的说教与惩罚,他可能会说'我不知道',这时先鼓励孩子,让他给予时间让他思考,在时间结束后再碰头解决问题,如果孩子没有思考,则继续给予时间,循环,知道该问题的解决
  • 孩子不做作业时,询问孩子为什么有家庭作业?一起决定做作业的时间,规定父母帮助孩子解决问题的时间,后面坚定的执行,让孩子理解、参与、制定规则跟限制(孩子可以决定花多少时间写作业,什么时候写作业,这些都是孩子自己的事情,需要孩子自己决定)
