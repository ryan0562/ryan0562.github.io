# 道枢大屏设计器 - 说明文档

# 简介

针对大屏业务提供可视化编辑器，结合模板库、模型超市，形成一套完整的可视化编辑工具，从而快速、高效的编辑出业务需要的大屏工程。

完整的工具链包括：工程库、模板库、模型超市、大屏编辑器。

# 特性

模板库、模型超市都有独立的仓库，可以独立部署、打包，并通过上传来接入使用

# 优势

  • 快速、高效搭建大屏工程
  • 统一模板、组件规范，解决不同组件、模板无法复用、使用成本高等问题
  • 统一化、规范化模型、模板
  • 易操作，学习成本非常低，贴合正常操作习惯。看一遍即能上手操作
  • 模板、模型超市解耦，形成单独的项目、按需上传

# 意义

  • 提高售前、产品快速展示大屏给客户观看，从而占据项目承接先机，提高成功率
  • 减少开发人员开发大屏的时间损耗、人员成本
  • 统一模板、组件规范，解决不同组件、模板无法复用、使用成本高等问题
  • 统一、规范的使用方式方法，便于管理，减少项目迁移以及交接的成本
  • 易操作，学习成本非常低，贴合正常操作习惯。看一遍即能上手操作
  • 模板、模型超市都可以通过上传方式载入到项目内，便于在公司内部开发完成后，直接上传到现场，不需要现场重新开发代码以及重新打包

# 名次释义

  • 工程：配置与数据的集合，大屏页面编辑后的页面产物。
  • 模板：配置大屏的标准化样式、布局等集合
  • 模型（组件）：布局容器内展示的业务模型，是主要的业务输出、展示手段
  • 场景：大屏内单独页面的统称，用来表示某些功能的集合页
  • 布局：场景内页面内容物位置、区域布置的统称
  • 属性面板：编辑器右侧属性配置区
  • 单元格：放置模型的基本单元区域
  • 布局面板：放置整个业务单元格的区域

  • 路由工具：Vue Router 3.5.1 (opens new window)
  • 状态管理：Vuex 3.6.2 (opens new window)
  • UI 框架：ElementUI 2.15.9 (opens new window)
  • CSS 预编译：Sass (opens new window) && Less (opens new window)
  • 接口通讯工具：Axios (opens new window)
  • 代码规范：Prettier (opens new window) + ESLint (opens new window)
