# 编辑器说明文档

  • 编辑器是编辑工程的可视化工具
  • 编辑器可以调用模板、模型拼装，从而搭配出大屏
  • 编辑器界面为上左中右结构：头部为标题、按钮功能区、左侧展示场景、中间是大屏展示区、右侧是属性配置区

# 使用说明

# 环境说明：

要求 node 版本为 ^16.0.0 建议使用 nvm 进行 node 版本管理。 nvm-windows (opens new window)

采用的依赖管理工具为 Yarn

# 常用命令

# 换源
yarn config set registry http://10.168.4.75:8081/repository/mti-npm/

# 安装
yarn install

# 启动
yarn dev

# 启动 Node 服务
yarn server

# 生产打包
yarn build

# 框架配置

  • 构建工具：vue-cli5 (opens new window) && webpack 5 (opens new window)
  • 前端框架： Vue 2.6.14 (opens new window)
  • 路由工具：Vue Router 3.5.1 (opens new window)
  • 状态管理：Vuex 3.6.2 (opens new window)
  • UI 框架：ElementUI 2.15.9 (opens new window)
  • CSS 预编译：Sass (opens new window) && Less (opens new window)
  • 接口通讯工具：Axios (opens new window)
  • 代码规范：Prettier (opens new window) + ESLint (opens new window)
  • 通用工具库：lodash (opens new window)
  • 伪数据拦截请求：mockjs (opens new window)
  • 页面进度条：nprogress (opens new window)
  • 数据持久化：vue-ls (opens new window)
  • 数据加密：crypto-js (opens new window)

# 功能配置

  • 动画库：animate.css (opens new window)
  • async/await报错：await-to-js (opens new window)异步处理等待包装器，便于处理异步报错
  • 时间、日期处理库：dayjs (opens new window)
  • DOM截图：dom-to-image-more (opens new window)
  • 可视化图表： echarts (opens new window)

# 功能说明

  1. 保存本地
    • 保存在本地页面缓存，保证刷新不会被清理
  2. 取消
    • 取消在编辑器的所有操作，并返回工程列表
  3. 预览
    • 预览当前配置的大屏界面，可以操作与交互
  4. 保存/发布
    • 将当前配置保存到数据库，并返回工程列表
  5. 布局
    • 对于场景内页面内容物位置、区域布置的统称
  6. 场景、场景列表
    • 场景：大屏内单独页面的统称，用来表示某功能的集合页
    • 场景列表：编辑器内左侧显示的列表，场景的集合。可以选中、调整布局、编辑名称、删除
    • 修改布局：修改当前场景的布局配置
    • 编辑场景名：修改当前场景名称，若是当前选中场景，会一同更改选中配置
    • 删除场景：删除选择的场景，并删除其内的全部数据与配置包括（布局数据、模型数据、配置数据等）
    • 新建场景：添加新的空白场景页面，后续需要选择布局以及填充模型与配置
  7. 属性面板
    • 在编辑视图中，有很多组件或者区域是可以配置自定义属性的，鼠标移上到会有黄色边框显示
    • 点击自定义配置的组件或者区域即可在右侧显示自定义配置的属性
    • 修改属性可实时在编辑视图中显示
    • 配置属性面板是在模板内根据业务需求预定义好
  8. 编辑视图（大屏展示区）
    • 实时观看配置的大屏界面
  9. 添加模型（组件）
    • 弹出模型超市弹窗，可以根据：组件类型、使用场景、模型名称等搜索条件选择模型
    • 选择模型后，按需加载模型资源
