# 模板库开发文档

  • 模板是大屏的基座，是预配置大屏的标准化样式、布局等集合
  • 模板能极大的减少配置流程，方便、快速的搭建大屏工程

# 环境说明：

要求 node 版本为 ^16.0.0 建议使用 nvm 进行 node 版本管理。 nvm-windows (opens new window)

采用的依赖管理工具为 Yarn

# 常用命令

# 换源
yarn config set registry http://10.168.4.75:8081/repository/mti-npm/

# 安装
yarn install

# 启动
yarn dev

# 打上传包
yarn lib

启动项目注意事项

  1. 注意，子模块(large-screen-modularity)不得提交，上传任何代码，有bug或者迭代，在编辑器git里修改，这里只读
  2. 子模块内如果没有代码，命令行输入git submodule update --init
  3. 如果节点不对，命令行git submodule update

# 开发启动流程

# 1. 更新large-screen-modularity子模块（编辑器）

  • 命令行输入git submodule update --init 或通过gitTortoise工具里的更新子模块

# 2. 注册全局link模块，以及引用

  • 在根目录运行npm link（注册全局link）,然后再运行npm link templates（使用刚刚注册的link，templates是包名）

# 3. 编辑器启动开发模板状态

// large-screen-modularity\src\store\index.js
 state: {
    // 是否开发模板
    developmentTemplates: true,
  },

上述修改后即可直接实时开发与调试，不用重新打包与上传。

# 全局注册模板

  1. 完成模板开发后，需要在packages/index.js内全局注册
import metas from './templateList.js';

import gaoxin329 from './gaoxin329/index.js'; // 高新模板
import huagong329 from './huagong329/index.js'; //化工模板

const components = [gaoxin329, huagong329]; // 添加到组件列表内

// 注册函数
const install = function (Vue, opts = {}) {
  components.forEach(component => {
    Vue.component(component.name, component);
  });
};

/* istanbul ignore if */
if (typeof window !== 'undefined' && window.Vue) {
  install(window.Vue);
}

export default {
  install,
  gaoxin329,
  huagong329
};

export const templateList = metas;
  1. 在packages/templateList.js注册 这里是为了开发调试所用

import huagong329 from './huagong329/src/meta.json';

const store = {
  gaoxin329,

# 说明

配置属性面板主要有两点，添加配置指令以及触发属性面板改变事件

  1. 配置指令：鼠标悬停的时候会有黄色边框，点击添加黄色边框等样式处理

Vue.directive('hasProps', {
  // 被绑定元素插入父节点时
  bind: (el, { value = {}, arg }, vnode) => {
    /* 非编辑状态返回 */
    if (router.currentRoute.query.edit !== '1') {
      return;
    }

    /* 处理样式 */
    el.addEventListener('mouseover', e => {
      e.stopPropagation();
      el.classList.add('hover');
    });
    el.addEventListener('mouseout', e => {
      e.stopPropagation();
      el.classList.remove('hover');
    });

    /* 添加固定样式 */
    el.classList.add('hasProps');
    
    /* 样式处理 */
    el.addEventListener('click', e => {
      e.stopPropagation();
      /* 删除所有dom选中样式 */
      const removeList = document.querySelectorAll('.hasProps.act');
      removeList.forEach(item => item.classList.remove('act'));

      // Vue.bus.$emit('changePanelForm', JSONRender(value.target, value.props));

      /* 添加选择样式 */
      el.classList.add('act');

    });
  }
});
  1. 触发属性面板改变事件 changePanelForm事件是已经注册在事件总线上的，后续逻辑以及配置都已集成，只需要返回配置JSON即可
