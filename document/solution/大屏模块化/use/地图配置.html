# 定制化开发

DANGER

定制化开发的代码只允许在模板或模型里开发

不允许在工程库里开发，会影响其他工程

一张图的实例已挂载到window上，通过这2个，可以直接自定义上图。

  • window.customMti （相当于gis的window.mtiMap）
  • window.customMap （相当于 new window.mtiMap.Map 的地图实例）
