# 工程库说明文档

  • 每一个大屏项目都是一个工程
  • 工程库是整个大屏设计器对外输出的产物
  • 通过发布链接，可以让用户直接观看与操作已经设计好的大屏界面

# 登录

登录分两种模式：编辑器模式、大屏模式。从那种模式登录的退出后就会跳转到哪种模式的登录页

  1. 编辑器模式：/editorLogin
  2. 大屏模式：/login

# 新建工程

创建新工程

# 编辑工程

在编辑器修改已有工程模型与数据,保存后的数据会覆盖选中的工程

# 复制工程

会复制选中的工程的数据、模型、模板、配置等，快速创建一个新的工程

# 查看发布

已发布大屏的展示链接，对外开放工程大屏。点击按钮后打开新窗口跳转到大屏展示界面。
