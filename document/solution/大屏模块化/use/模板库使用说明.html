# 工程库说明文档

  • 模板是大屏的基座，是预配置大屏的标准化样式、布局等集合
  • 模板能极大的减少配置流程，方便、快速的搭建大屏工程

# 上传模板

上传完成后会立刻刷新模板库列表，展示已上传的模板。

# 模板使用

  • 一般都是在视图区选择需要编辑的组件，然后再在右侧属性面板编辑
  • 可编辑的组件都会在鼠标移动上的时候显示黄色的边框
