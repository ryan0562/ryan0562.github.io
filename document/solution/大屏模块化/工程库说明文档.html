# 环境说明：

要求 node 版本为 ^16.0.0 建议使用 nvm 进行 node 版本管理。 nvm-windows

采用的依赖管理工具为 Yarn yarn 安装文档

# 常用命令

# 换源
yarn config set registry http://10.168.4.75:8081/repository/mti-npm/

# 安装
yarn install

# 启动
yarn dev

# 启动 Node 服务
yarn server

# 生产打包
yarn build
