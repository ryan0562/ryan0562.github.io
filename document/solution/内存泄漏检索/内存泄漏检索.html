# 内存泄漏检索

  1. vue.bus.on是否在在销毁钩子上有off
  2. addEventListener是否在销毁钩子上有removeEventListener
  3. 定时器是否在销毁钩子上有销毁
  4. 检查是否存在防抖、节流或者其他闭包的函数在初始化时调用
