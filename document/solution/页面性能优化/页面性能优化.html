# 页面性能优化

# sourcemap

如果线上版本没去掉sourcemap的话，js体积会变的非常大，会导致下载和解析都变慢

# webpack默认会加上sourcemap吗？

  • dev环境的包，默认会加
  • prd环境的包，默认会去掉sourcemap

# 如何确定js内是否有sourcemap？

随意打开一个js文件，搜索关键字 “ sourceMappingURL ”。 如果看到类似以下的代码，那就说明里面含了sourcemap
