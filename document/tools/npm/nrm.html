    # 私有源发布

    如果你的公司有私域npm管理源，或者平时喜欢用淘宝源，推荐使用 nrm 进行切换：

    npm i nrm -g
    1. 查看源: nrm ls
    2. 添加源: nrm add name http//:xxx.xxx.xxx.xxx:4873/
    3. 删除源: nrm del name
    4. 使用指定源: nrm use npm
